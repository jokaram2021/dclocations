
## Overview


The API handles:

- User registration
- User login
- JWT bearer token authentication
- Location retrieval
- Featured locations
- Location create, update, and delete actions for admins
- Favorites management for logged-in users
- External read-only Pokémon Type API integration
- Swagger API documentation

The API is built with ASP.NET Core Web API and uses a MySQL database running in a Docker container.

---

## API Scope

### Included

- User registration
- User login
- JWT token generation
- Location retrieval
- Featured location retrieval
- Location detail retrieval
- Location view tracking
- Admin location creation
- Admin location editing
- Admin location deletion
- Favorites retrieval
- Add favorite
- Remove favorite
- External Pokémon Type API call for Energy Match
- Swagger documentation



---

## Base URL

When running locally with Docker, the API is available at http://localhost:5153
